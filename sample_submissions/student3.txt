Factorial Function Implementation

I will implement a factorial function in Python that handles various edge cases.

def factorial(n):
    if n < 0:
        return "Error: Cannot calculate factorial of negative number"
    if n == 0:
        return 1
    if n == 1:
        return 1
    
    result = 1
    for i in range(1, n + 1):
        result = result * i
    return result

This implementation:
1. Handles negative numbers by returning an error message
2. Correctly returns 1 for factorial of 0 and 1
3. Uses a simple loop to calculate factorial for positive numbers
4. Is easy to understand and follow

Test cases:
- factorial(5) = 120
- factorial(0) = 1
- factorial(-1) = "Error: Cannot calculate factorial of negative number"
